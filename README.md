# 💪 Gym Workout Tracker

A modern, mobile-first web app to track your gym workouts بسهولة وبشكل احترافي.

🔗 **Live Demo:**
https://azmyehab21.github.io/Gym-Workout-Tracker/

---

## 🚀 Features

* 🏋️ Add workouts (Exercise, Weight, Reps, Sets, Date)
* 📊 Dashboard with:

  * Total workouts
  * Last workout
  * Heaviest weight
* ✏️ Edit & Delete workouts
* 🔍 Search by exercise name
* 📅 Filter by date
* 💾 Data saved باستخدام Local Storage
* 🔔 Toast notifications (Add / Update / Delete)
* 🎨 Modern UI (Dark mode + Mobile-first)
* 📱 Fully responsive (optimized for phones)
* 🍔 Hamburger menu navigation

---

## 🧠 فكرة المشروع

تطبيقات تتبع التمارين بتساعدك تسجل تقدمك وتتابع الأداء بدل الطرق التقليدية زي الورق أو النوت، وده بيساعدك تطور مستواك بشكل أفضل وتفهم تقدمك مع الوقت ([WorkoutLogs][1])

## 🛠️ Tech Stack

* HTML5
* CSS3

* Local Storage API

## ⚙️ How to Use

1. افتح الموقع من اللينك
2. اضغط Add Exercise
3. سجل بيانات التمرين
4. راجع التمارين من صفحة Workouts
5. تابع تقدمك من Dashboard
